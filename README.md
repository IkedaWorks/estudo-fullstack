# Full-Stack Development Journey 🚀

Repositório centralizado para o desenvolvimento de soluções web, unindo interfaces dinâmicas no front-end a arquiteturas sólidas de back-end. Este espaço funciona como um laboratório pessoal para consolidar conceitos de engenharia de software, manipulação de dados e integração de APIs.

---

## 📂 Estrutura do Repositório

O ecossistema é dividido modularmente para isolar as responsabilidades de cada camada:

* **`front-end/`**: Projetos focados em experiência do usuário (UX/UI), consumo de APIs, manipulação de estado e estilização moderna.
* **`back-end/`**: Desenvolvimento de APIs, lógica de negócios, segurança, conexão com bancos de dados e ecossistemas server-side (foco em PHP e arquiteturas de servidores).

---

## 🛠️ Tecnologias & Objetivos de Estudo

| Camada | Tecnologias Core | Foco de Aprendizado |
| :--- | :--- | :--- |
| **Front-End** | HTML5, CSS3, JavaScript (ES6+) | Componentização, DOM, Consumo de APIs Assíncronas (Fetch/Axios). |
| **Back-End** | PHP, POO, MySQL / PostgreSQL | Arquitetura MVC, Manipulação de requisições, Segurança (Sanitização e Autenticação). |
